## 代码块

支持语法高亮，加上语言名即可：

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))  # 55
```

```javascript
const greet = (name) => `Hello, ${name}!`
console.log(greet('World'))
```

### 高亮特定行

```python {2,4}
def add(a, b):
    # 这一行被高亮
    result = a + b
    return result  # 这一行也被高亮
```
